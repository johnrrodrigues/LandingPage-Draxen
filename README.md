## Sobre o Projeto

Landing page do **OneHelp**, sistema web de gestão de ONGs de adoção animal, desenvolvido pela equipe Draxen Systems como projeto interdisciplinar do curso de Desenvolvimento de Software Multiplataforma (DSM).

O projeto nasceu de uma parceria com a ONG **Bicho Carente** de Leme, SP, com o objetivo de centralizar e organizar o processo de adoção animal — substituindo planilhas e anotações manuais por uma plataforma intuitiva e eficiente.
