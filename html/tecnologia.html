Tecnologias Utilizadas

Ferramentas e linguagens base para o desenvolvimento do sistema.

  • GitHub

    Controle de versão e colaboração no repositório de código.

  • Python

    Linguagem de programação principal utilizada no back-end.

  • Django

    Framework web utilizado para estruturar o back-end e as rotas.

  • React

    Biblioteca JavaScript para a construção da interface de usuário (front-end).

  • MySQL

    Banco de dados relacional para armazenamento estruturado das informações.

  • MongoDB

    Banco de dados NoSQL utilizado para dados não estruturados ou mais flexíveis.

  • HTML

    Linguagem de marcação para a estruturação dos componentes da página.

  • CSS

    Folhas de estilo para o layout e apresentação visual da interface.
